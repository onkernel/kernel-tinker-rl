# Agent Auth - Login Discovery Example

This example demonstrates training an agent to find login pages and identify required input fields.

## Overview

The Agent Auth agent:

1. Navigates to a website
2. Finds the login or registration page
3. Identifies the input fields required to begin the login process
4. Reports the fields using a custom `request_inputs` action

## Custom Action

This example adds a custom terminal action:

```python
@dataclass
class RequestInputsAction(Action):
    """Request input fields from the user."""

    fields: list[InputField]

    action_type: ClassVar[str] = "request_inputs"
    is_terminal: ClassVar[bool] = True
```

## Task Data

LLM-generated by browsing to the top 1000 sites on the internet and generating realistic tasks related to logging in or signing up.

## Running

```bash
# Run agent on a single domain
uv run python -m scripts.run_agent \
    --url https://github.com \
    --task "Navigate to github.com and find the login page. Identify the input fields."
```
